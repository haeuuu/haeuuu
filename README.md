#### 🍋 **다함께 으쌰으쌰 팀프로젝트**
* `RFM analysis` : 어떤 고객이 가치가 높은 고객일까? 지표를 만들고 분류합니다.
* `NER-for-Keyword-Extraction` : 한국사에선 인명이 중요하지! 개체명을 예측하여 중요 키워드를 추출합니다.

#### 🍈 **추천 시스템 구현해보자!**
* `Recsys-paper-review` : 추천 시스템 논문을 공부합니다.
* `Auto-encoder`, `Word2Vec`, `Matrix Factorizatin(ALS)` based recommendation system (for playlist continuation)
* `Title-based-playlist-generator` : 듣고 싶은 플레이 리스트의 제목만 입력하면, 적절한 노래를 찾아 플레이 리스트를 생성합니다.

#### 🍊 **귀찮은건 싫어! 내가 쓰려고 만든 프로그램**
* `Melon-ply-generator` : (곡명,가수) 리스트를 입력하면, 자동으로 멜론에 플레이 리스트를 생성합니다.
* `html-conveter-for-tistory` : html 파일을 tistory에 업로드할 때 깨지지 않도록 형식에 맞게 변환합니다.
* `pdf-to-image` : pdf를 image로 변환하여 저장합니다.
